V2.01 : 2020/01/27 (初版)

スキャナーで読み取ったイメージファイルから文字を読み取りテキストファイルを作成します。
(以前にアップしたVBOCRに誤認識対応機能を付加したものです)
イメージファイルは "Windows Fax と スキャン" で スキャナーからイメージファイルを得ることができます
インストールは zip ファイルを展開し、SetupVBOCR2.msi をダブルクリックしてください

要件：文書読み取り機能は "Microsoft OCR Library for Windows" を使用しています
      Windows10 Novenber Update(2015/11:Version 1511)にアップグレードが必要です

MSVS2017Communityプロジェクトソースは GitHub sakachin2/VBOCR2 にあります
